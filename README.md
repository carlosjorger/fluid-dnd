Fluid DnD is a [**fluid**, **agnostic** and **versatile** drag and drop
library for lists; with support for **Vue**, **React** and **Svelte**](https://fluid-dnd.netlify.app/). It's a **lightweight** tool ~8 Kb (gzip) with no depenencies.
